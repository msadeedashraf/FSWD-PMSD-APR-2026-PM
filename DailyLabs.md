# 📘 IT Infrastructure & Software Development Labs (5-Day System)

This lab system is designed to help students:

* Understand how IT systems work
* Learn real-world terminology through research
* Connect infrastructure to how software is built
* Think through real scenarios like engineers

---

# 🔁 Daily Lab Structure

Each lab follows:

1. **Explore** (Research concepts)
2. **Explain** (Simple + technical explanation)
3. **Apply** (Real-world thinking)
4. **Build** (Structured output)
