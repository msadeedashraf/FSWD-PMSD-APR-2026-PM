# 🧠 Case Study 1 — FTP to Cloud Migration

## Scenario

A company currently:

* Uses FTP servers for file transfer
* Uses VPN for external vendors
* Faces delays and security concerns

They want:

* A modern cloud-based web platform

## Questions

1. Break this project into **SDLC phases** and explain each phase in this context
2. Choose **Centralized vs Distributed architecture** and justify
3. Should this use **Waterfall or Agile**? Why?
4. Identify **3 risks** (technical + business)
5. Which tool would you use: Asana or Jira? Why?

# 🧠 Case Study 2 — Distributed Database Failure

## Scenario

A company uses:

* Distributed database across 3 regions
* Data replication (push/pull)

Problem:

* Data inconsistency between regions

## Questions

1. Which **SDLC phase failed** and why?
2. Would **Waterfall** have worked here? Explain
3. What changes would you make in the **Design phase**?
4. What **testing strategy** should have been used?
5. What type of **project planning** is critical for distributed systems?
